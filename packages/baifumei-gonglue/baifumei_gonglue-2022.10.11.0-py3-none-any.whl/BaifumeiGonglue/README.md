# 白富美攻略 - 欧阳浮夸

## 下载

### Docker

```
docker pull apachecn0/baifumei-gonglue
docker run -tid -p <port>:80 apachecn0/baifumei-gonglue
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install baifumei-gonglue
baifumei-gonglue <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g baifumei-gonglue
baifumei-gonglue <port>
# 访问 http://localhost:{port} 查看文档
```
