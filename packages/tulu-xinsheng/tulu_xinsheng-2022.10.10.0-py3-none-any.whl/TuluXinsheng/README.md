# 吐露心声

## 下载

### Docker

```
docker pull apachecn0/tulu-xinsheng
docker run -tid -p <port>:80 apachecn0/tulu-xinsheng
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install tulu-xinsheng
tulu-xinsheng <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g tulu-xinsheng
tulu-xinsheng <port>
# 访问 http://localhost:{port} 查看文档
```
