## 版本

v0.2.2a1

⚠ 适配nonebot2-2.0.0b3+

## 安装

通过`pip`或`nb`安装。

## 功能

掷骰！扔出指定个数的多面骰子；支持群聊与私聊。

## 命令

掷骰子：[rd/roll/掷骰] [x]d[y]，掷出x个y面的骰子，并返回点数。
