# 男女内参

## 下载

### Docker

```
docker pull apachecn0/nannv-neican
docker run -tid -p <port>:80 apachecn0/nannv-neican
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install nannv-neican
nannv-neican <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g nannv-neican
nannv-neican <port>
# 访问 http://localhost:{port} 查看文档
```
