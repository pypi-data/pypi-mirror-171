### Docker

```
docker pull apachecn0/mogui-zixunshi-jinghuatie-huizong
docker run -tid -p <port>:80 apachecn0/mogui-zixunshi-jinghuatie-huizong
# 访问 http://localhost:{port} 查看文档
```
