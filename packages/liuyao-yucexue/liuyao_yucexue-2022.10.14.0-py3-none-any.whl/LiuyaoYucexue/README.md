# 六爻预测学

## 下载

### Docker

```
docker pull apachecn0/liuyao-yucexue
docker run -tid -p <port>:80 apachecn0/liuyao-yucexue
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install liuyao-yucexue
liuyao-yucexue <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g liuyao-yucexue
liuyao-yucexue <port>
# 访问 http://localhost:{port} 查看文档
```
