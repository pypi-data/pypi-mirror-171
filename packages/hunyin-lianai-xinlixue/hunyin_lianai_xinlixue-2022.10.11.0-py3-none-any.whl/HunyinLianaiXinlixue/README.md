## 下载

### Docker

```
docker pull apachecn0/hunyin-lianai-xinlixue
docker run -tid -p <port>:80 apachecn0/hunyin-lianai-xinlixue
# 访问 http://localhost:{port} 查看文档
```

```
pip install hunyin-lianai-xinlixue

# 访问 http://localhost:{port} 查看文档
```

# 访问 http://localhost:{port} 查看文档
