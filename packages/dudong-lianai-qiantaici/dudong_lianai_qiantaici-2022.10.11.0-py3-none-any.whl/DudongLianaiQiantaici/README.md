### Docker

```
docker pull apachecn0/dudong-lianai-qiantaici
docker run -tid -p <port>:80 apachecn0/dudong-lianai-qiantaici
# 访问 http://localhost:{port} 查看文档
```

```
dudong-lianai-qiantaici <port>
# 访问 http://localhost:{port} 查看文档
```
