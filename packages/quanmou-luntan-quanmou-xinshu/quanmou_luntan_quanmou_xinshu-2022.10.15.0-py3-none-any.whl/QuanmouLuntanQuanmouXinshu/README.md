# 权谋论坛：权谋心术

## 下载

### Docker

```
docker pull apachecn0/quanmou-luntan-quanmou-xinshu
docker run -tid -p <port>:80 apachecn0/quanmou-luntan-quanmou-xinshu
# 访问 http://localhost:{port} 查看文档
```
