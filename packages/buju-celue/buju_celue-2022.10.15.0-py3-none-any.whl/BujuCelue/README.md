# 布局策略

## 下载

### Docker

```
docker pull apachecn0/buju-celue
docker run -tid -p <port>:80 apachecn0/buju-celue
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install buju-celue
buju-celue <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g buju-celue
buju-celue <port>
# 访问 http://localhost:{port} 查看文档
```
