# 权谋论坛：权谋转机

## 下载

### Docker

```
docker pull apachecn0/quanmou-luntan-quanmou-zhuanji
docker run -tid -p <port>:80 apachecn0/quanmou-luntan-quanmou-zhuanji
# 访问 http://localhost:{port} 查看文档
```
