# 未知之门

## 下载

### Docker

```
docker pull apachecn0/weizhi-zhimen
docker run -tid -p <port>:80 apachecn0/weizhi-zhimen
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install weizhi-zhimen
weizhi-zhimen <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g weizhi-zhimen
weizhi-zhimen <port>
# 访问 http://localhost:{port} 查看文档
```
