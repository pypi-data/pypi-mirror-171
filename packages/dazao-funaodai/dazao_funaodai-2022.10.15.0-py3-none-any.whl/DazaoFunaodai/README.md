# 打造富脑袋

## 下载

### Docker

```
docker pull apachecn0/dazao-funaodai
docker run -tid -p <port>:80 apachecn0/dazao-funaodai
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install dazao-funaodai
dazao-funaodai <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g dazao-funaodai
dazao-funaodai <port>
# 访问 http://localhost:{port} 查看文档
```
