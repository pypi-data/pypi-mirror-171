# 子弹笔记

## 下载

### Docker

```
docker pull apachecn0/zidan-biji
docker run -tid -p <port>:80 apachecn0/zidan-biji
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install zidan-biji
zidan-biji <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g zidan-biji
zidan-biji <port>
# 访问 http://localhost:{port} 查看文档
```
