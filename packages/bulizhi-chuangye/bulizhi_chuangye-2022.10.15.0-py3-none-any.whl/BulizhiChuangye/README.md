# 不离职创业

## 下载

### Docker

```
docker pull apachecn0/bulizhi-chuangye
docker run -tid -p <port>:80 apachecn0/bulizhi-chuangye
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install bulizhi-chuangye
bulizhi-chuangye <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g bulizhi-chuangye
bulizhi-chuangye <port>
# 访问 http://localhost:{port} 查看文档
```
