# 习惯的力量

## 下载

### Docker

```
docker pull apachecn0/xiguande-liliang
docker run -tid -p <port>:80 apachecn0/xiguande-liliang
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install xiguande-liliang
xiguande-liliang <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g xiguande-liliang
xiguande-liliang <port>
# 访问 http://localhost:{port} 查看文档
```
